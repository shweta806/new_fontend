Guideline for Setup

Step 1: Set Up VirtualBox

Download and Install: Get the latest version of VirtualBox from the official website and install it on your system.
Configure Settings: Open VirtualBox and adjust preferences like the default machine folder and network settings for optimal performance.

Step 2: Create Virtual Machines

Create VMs: Click "New" in VirtualBox to create multiple VMs, specifying the OS type and version.
Allocate Resources: Assign appropriate CPU, memory, and storage to each VM, ensuring the primary VM has enough resources to manage the others.
Designate Primary VM: Choose one VM as the primary controller for managing network connections.

Step 3: Configure Network Connections:

Primary VM Setup:In the primary VM's settings, select "Bridged Adapter" or "NAT" to connect to the internet.
Private Network for Other VMs:Set the other VMs to "Internal Network" or "Host-Only Adapter" for private communication.
Connect VMs: Ensure all VMs can route traffic through the primary VM.

Step 4: Test Connectivity:

Ping Test: Use the ping command in each VM to check connectivity with the primary VM.
Internet Access Check: Verify that the primary VM can access the internet and that other VMs can connect through it.
Troubleshoot: If issues arise, check network settings and firewall configurations.
